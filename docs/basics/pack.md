# 包装类
- Java语言提供了八种基本类型，六种数字类型（四个整数型，两个浮点型），一种字符类型，还有一种布尔型。
1. 整数：包括int,short,byte,long，初始值为0
2. 浮点型：float,double，初始值为0.0
3. 字符：char，初始值为空格，
4. 布尔：boolean，初始值为false  
| 基本类别 | 大小 | 最小值 | 最大值 |
| ------- | ---- | ----- | ------ |
| boolean | ---- | ----- | ------ |
| char    | 16bit | Unicode 0 | Unicode 2^16-1 |
